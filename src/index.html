How it works

Model Context Protocol

Bring your tables into the AI tools you already use

TypeTable speaks MCP. Connect Claude Code, Cursor, or any MCP‑compatible client and let your agents read and update your tables — your data becomes structured memory any AI can use.

  • Real records, not notes. Agents list tables, query rows, and add or edit records — against the same validated data you use in chat.
  • Your rules still apply. Types, required fields, and permissions hold whether a person or an agent writes.
  • One secure token. Create an API token in Settings and connect over Streamable HTTP. Revoke it any time.
